### アプリ概要
「あれどこだっけ？」をなくす収納管理アプリ 
「どこに何をしまったか分からない」を解決する、写真ベースの収納管理アプリです。
「場所」と「しまう物」を撮影し、紐づけて管理します。
 名前付けや細かい分類を行わず、写真だけで直感的に探せます。 
 
### 解決する課題
探し物をするとき、人は「名前」ではなく「見た目（イメージ）」で探します。
しかし文字ベースの管理では「どこに分類したのか思い出せない」「どう分類したか忘れた」といった問題が発生します。
本アプリは「見れば分かる」ことで、記憶に頼らない管理を実現します。

### ターゲットユーザー
・20〜40代のスマホ利用者 
・物の管理をシンプルにしたい人  
・「あれどこだっけ？」が日常的に発生する人 
※自分自身の課題から着想しています

### 主な機能
収納場所の写真を登録（カメラ対応） 
アイテムの写真を登録（カメラ対応） 
場所の一覧表示 
アイテムの確認チェック機能 
画像のモーダル表示
